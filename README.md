# CloudNav云导航

一个基于Cloudflare Workers构建的现代化网站导航工具，帮助您整理和快速访问常用网站。

## 功能特点

- 📚 网站分组管理：按类别整理您的网站
- 🔍 快速搜索：快速找到您需要的网站
- 🏷️ 自定义标签：为网站添加标签进行分类
- 📱 响应式设计：支持各种设备访问
- 🌓 暗色/亮色模式：根据您的喜好切换主题
- 📊 访问统计：了解您的访问习惯
- 🔄 备份与恢复：保护您的数据安全
- 👥 多用户管理：支持多用户协作

## 技术栈

- 前端：React, TypeScript, TailwindCSS, DaisyUI
- 后端：Cloudflare Workers, Cloudflare KV, Cloudflare D1
- 构建工具：Vite
- 状态管理：Zustand

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建项目

```bash
npm run build
```

### 预览构建结果

```bash
npm run preview
```

### 部署到Cloudflare Workers

```bash
npm run deploy
```

```
│   ├── styles/      # 样式

│   └── utils/       # 工具函数

```
